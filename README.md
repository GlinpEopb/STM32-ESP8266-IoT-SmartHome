# STM32-ESP8266-IoT-SmartHome
🌐 STM32-based IoT Smart Home System | ESP8266 WiFi + OneNET Cloud + MQTT

A complete microcontroller IoT project that realizes real-time environmental monitoring and remote control for smart homes.

### ✨ Core Features
- **Real-time Monitoring**: Temperature & Humidity (DHT11), Light Intensity (BH1750)
- **Wireless Communication**: ESP8266 WiFi module with AT commands and MQTT protocol
- **Cloud Integration**: OneNET cloud platform for data storage, visualization and remote control
- **Local Display**: 0.96" OLED screen for on-site data display
- **Remote Control**: Mobile APP / Web control of devices (e.g. lighting, alerts)
- **Hardware**: STM32F103C8T6 + custom PCB designed in 嘉立创 (JLCEDA)

### 🛠️ Technologies
- **MCU**: STM32F103C8T6 (Keil + Standard Library)
- **WiFi**: ESP8266 (UART + MQTT)
- **Sensors**: DHT11, BH1750
- **Display**: SSD1306 OLED (I2C)
- **IoT Platform**: OneNET + MQTT
- **PCB Design**: 嘉立创 EDA
